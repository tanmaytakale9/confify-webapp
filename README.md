# 💜 Confify

Confify is a modern anonymous messaging platform where users can create a personal link and receive honest messages from friends while protecting privacy.

## ✨ Features

- 🔒 Anonymous messaging
- 👤 Personal shareable profile link
- 🛡️ AI toxicity filtering
- 🚫 Spam & rate limiting
- 📊 User analytics dashboard
- ❤️ Message reactions
- 📱 Fully responsive design
- 🌙 Modern UI

## 🛠️ Tech Stack

- Next.js 16
- React
- MongoDB Atlas
- Mongoose
- CSS
- Vercel
